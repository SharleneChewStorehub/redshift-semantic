## Environments
* TypeScript 3.5.1
* node 10.15.1
* yarn 1.13.0
* Visual Studio Code (Recommond IDE) and [TSLint][tslint], [Prettier][vcprettier]

## Getting Started
`cp .env.example .env`

### build ts
`yarn build`

### start app
`yarn & yarn start`

### test project
1. `yarn test`
2. test with coverage `yarn test:coverage`
